   2. mint.js               - سك NFT واحد

   4. setBaseURI.js         - ضبط رابط الميتاداتا
   5. set-royalty.js        - ضبط نسبة العمولات
   6. upload-metadata.js    - رفع إلى IPFS
   7. check.js              - فحص بيانات العقد

✅ ملفات الميتاداتا (Metadata Files) - 3 NFTs
   - metadata/1.json: KingShield #1 (Legendary)
   - metadata/2.json: KingShield #2 (Epic)
   - metadata/3.json: KingShield #3 (Rare)
   - الحجم الكلي: ~30 سطر JSON
